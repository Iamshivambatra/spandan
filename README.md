# Spandan - Clinical Records Management System

A web-based clinical records management system for healthcare providers and nurses to manage patient information, clinical notes, treatment plans, and vitals.

## Features

- **Clinical Records**: Comprehensive patient clinical records capture
- **Treatment Plans**: Track treatment continuation and medical interventions
- **Vitals Monitoring**: Record and monitor patient vital signs
- **Nurse Records**: Dedicated interface for nursing documentation
- **Doctor Records**: Dedicated interface for physician documentation
- **Investigation Logs**: Track and manage patient investigations
- **Video Consultation History**: Record and retrieve video consultation sessions
- **Discharge Documentation**: Complete discharge summaries and instructions

## Project Structure

- `index.js` - Main JavaScript entry point
- `IMG NBSU CLINICAL RECORD.html` - Main clinical record interface
- `IMG NBSU CLINICAL RECORD EDITABLE.html` - Editable clinical record interface
- `discharge.html` - Discharge summary form
- `img*_nurse.html` - Nurse-specific documentation forms
- `img*_doctor.html` - Doctor-specific documentation forms
- Additional specialized forms for investigations, vitals, and consultations

## Getting Started

1. Clone this repository
2. Open `index.html` or any form file in your web browser
3. Fill in patient data and clinical information
